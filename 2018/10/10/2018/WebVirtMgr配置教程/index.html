WebVirtMgr安装教程

WebVirtMgr是一个基于libvirt的Web界面，用于管理虚拟机，而且允许创建和配置新域，并调整域的资源分配，适用于网络安全实验模拟。

安装环境

  • 系统：Ubuntu Server 14.04

确定 CPU 支持 KVM

KVM 依赖于 x86 虚拟化扩展。为了检查兼容性，请运行命令

  • egrep -c '(vmx|svm)' /proc/cpuinfo

检查 CPU 是否支持 KVM

如果输出类似于以下数值(大于0)，那么 CPU 就支持 KVM；否则，CPU 不支持 KVM。

如果是0说明你的CPU当前不支持虚拟化 。
